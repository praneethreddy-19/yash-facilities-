## 🚀 Deploy on Vercel
1. Import this repo at [vercel.com/new](https://vercel.com/new)
2. Select **Other** as Framework Preset
3. Click **Deploy** — no build step required

## 💻 Run Locally
```bash
# Using Python
python -m http.server 8000
# Open http://localhost:8000
```
